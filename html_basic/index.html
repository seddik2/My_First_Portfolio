Introduction

This page is a simple example of how to use various HTML elements. We will include headings of different levels, multiple paragraphs, and a clickable image. HTML, or HyperText Markup Language, is the standard language for creating web pages.

Headings and Paragraphs

Headings

Headings are used to create a hierarchical structure on your web page. There are six levels of headings in HTML, from <h1> to <h6>. The most important heading is <h1>, and the least important is <h6>.

Paragraphs

Paragraphs are used to group related sentences together. In HTML, paragraphs are created with the <p> tag. Each paragraph will appear on a new line and will have some space between it and the previous paragraph.

Conclusion

This example demonstrates the basic structure of an HTML document, including headings, paragraphs, and a clickable image.
